# ⚙ 주요 기능
### 🏠 메인 페이지
- 제작한 3개의 페이지에 대한 정보 및 이동할 수 있는 기능
### ✨ 여러 효과 확인 페이지
- 버튼 클릭 시 이미지가 바뀌는 갤러리
- 지구본 형태의 효과가 적용된 버튼
- 이미지에 다가가면 글자가 보이는 효과
### 💻 개인 노트북 페이지
- 개인적으로 가지고있는 노트북 정보를 모바일에서도 편히 볼 수 있는 기능 
### ☀ 계절 안내 페이지
- 사계절에 대한 정보를 제공
